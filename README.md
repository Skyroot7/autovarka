# Автоварка - Інтернет-магазин автомобільних мультиварок

Сучасний інтернет-магазин для продажу автомобільних мультиварок 12-24V.

## 🚀 Функціонал

- ✅ Мультиязычный интерфейс (UA, RU, EN, PL, DE)
- ✅ Каталог товаров с фильтрацией и сортировкой
- ✅ Корзина покупок с локальным хранением
- ✅ Детальные страницы товаров
- ✅ Страницы: Главная, Товары, О нас, Контакты
- ✅ Форма обратной связи
- ✅ Адаптивный дизайн
- ✅ SEO оптимизация
- ✅ Профиль пользователя (базовая структура)

## 🛠️ Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **next-intl** - мультиязычность
- **Zustand** - управление состоянием
- **Framer Motion** - анимации
- **Heroicons** - иконки

Сайт поддерживает 5 языков:
- 🇺🇦 Украинский (uk) - по умолчанию
- 🇷🇺 Русский (ru)
- 🇬🇧 Английский (en)
- 🇵🇱 Польский (pl)
- 🇩🇪 Немецкий (de)

## ⚙️ Конфигурация

### Email настройки

Создайте файл `.env.local`:

```
EMAIL_USER=info@autovarka.com.ua
EMAIL_PASS=ваш_пароль_приложения
```

### Telegram уведомления

Для настройки уведомлений в Telegram добавьте бота токен в `.env.local`.

## 📝 Товары

Товары хранятся в `lib/products.ts`. Добавьте свои товары туда с полными переводами на все языки.

## 💡 Разработка

Для добавления новых страниц:
1. Создайте файл в `app/[locale]/название/page.tsx`
2. Добавьте переводы в `messages/*.json`
3. Добавьте ссылку в `components/Header.tsx`

Для добавления товаров:
1. Добавьте объект в массив `products` в `lib/products.ts`
2. Укажите все переводы названий и описаний
3. Добавьте изображения в `public/images/`
